# TokIntel – CI/Visual Setup (Cursor One‑Shot)

Obiettivo: Preparare un nuovo clone per lavorare con ecosistema CI/Visual e badge Docs Ready.

## Passi automatici
1) Apri un terminal in Cursor e lancia:
```bash
git pull
make install-hooks
make ci-visual-refresh
make docs-ready
make docs-check
```

2) Verifica lo smoke:
```bash
make e2e-smoke
```

3) Commit & push:
```bash
git add -A
git commit -m "chore: bootstrap CI/Visual ecosystem on fresh clone"
git push
```

## Troubleshooting in Cursor
- Se vedi `missing separator`, sostituisci gli spazi con TAB nelle ricette Makefile.
- Se alcuni script non esistono, i target li saltano con messaggi "ℹ️". Aggiungili quando disponibili o ignora.
- Se il badge non cambia, rilancia:
```bash
make docs-ready && git add docs/status.json README.md && git commit -m "docs: refresh Docs Ready badge" && git push
```
